Welcome to Phasemix.

Phasemix builds a blog website in 3 simple steps:

  1. Edit
  2. Compose
  3. Publish
  1. Edit your content in any tool that exposes their data through an API such as Tumblr, Wordpress. 2 Compile new HTML files using phasemix generators. 3. Publish the generated HTMLs.

Right now Phasemix just supports Tumblr as a content source and Google Web Starter Kit as the template to generate the new htmls.
